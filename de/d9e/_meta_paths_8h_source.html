15 /// \brief Absolute organization-level ProgramData path.
16 /// \details
17 /// Returns the absolute, normalized path to:
18 /// [ProgramData]/[OrganizationName]
19 /// Resolution order:
20 /// 1) Windows API (SHGetKnownFolderPath with FOLDERID_ProgramData)
21 /// 2) Environment variable fallback: PROGRAMDATA
22 /// Then:
23 /// - Verifies the base folder exists (fatal if missing).
24 /// - Ensures the organization subdirectory exists (creates it; fatal if it fails).
25 /// - Canonicalizes when possible (resolves junctions/symlinks); otherwise cleans the absolute path.
26 /// - Converts separators to forward slashes; UNC is returned as //server/share/....
27 ///
28 /// The result is cached in a function-local static and reused on subsequent calls.
29 ///
30 /// \return Reference to a process-lifetime QString with the absolute path using forward slashes.
31 /// \post The returned reference remains valid for the lifetime of the process.
32 /// \note
33 /// - Initialized using `Q_CONSTRUCTOR_FUNCTION` during application startup to ensure availability
34 /// and to report any fatal errors early.
35 /// - Windows-only: Paths use forward slashes for Qt friendliness; call
36 /// QDir::toNativeSeparators(...) before passing to Win32 APIs.
37 /// - Multi-Thread: Safe, first-call initialization is thread-safe (C++11). Subsequent calls are lock-free.
38 /// \warning On unrecoverable failure (cannot locate or create the required directory),
39 /// a native dialog is shown and the process is terminated (no return).
40 /// \par Example
41 /// \code
42 /// // Build a file path within the org folder (Qt accepts forward slashes)
43 /// const QString orgDir = npi::app::orgProgramDataDir();
44 /// const QString file = QDir(orgDir).filePath("global.log");
45 ///
46 /// // When calling Win32, convert once:
47 /// const QString native = QDir::toNativeSeparators(orgDir);
48 /// // CreateDirectoryW(reinterpret_cast<LPCWSTR>(native.utf16()), nullptr);
49 /// \endcode
50 const QString& orgProgramDataDir() noexcept;
