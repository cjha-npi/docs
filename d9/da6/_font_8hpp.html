Detailed Description

Thread-safe font utility for application and fixed-width fonts.

This header defines npi::Font, a small utility that centralizes font handling:

  • Maintains a global application font and a global fixed-width font.
  • Provides style flags (bold, italic, underline, overline, strikeout) and capitalization.
  • Supports absolute and relative point sizes and computes effective sizes.
  • Caches QFont and its metrics (QFontMetrics, QFontMetricsF) keyed by style+size.
  • Uses per-family mutexes to guard static state and caches.
Note
GUI thread requirement: Any interaction with QWidget or QApplication must occur on the GUI thread. This class posts updates using queued connections where needed.
Warning
Ordering note for SetAppFont_ and SetFixFont_: These methods update internal state and clear caches under a mutex, unlock, and then perform GUI updates (qApp->setFont or posting QWidget::update()). This introduces a brief window where cache state and GUI state may not be perfectly in sync. This is acceptable in practice and avoids holding a lock across GUI calls.
