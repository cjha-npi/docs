Detailed Description

Helper for StageWindow class. A RAII guard that restores a widget's window opacity on scope exit.

Use when you temporarily set a target widget's opacity to 0.0 before an animation is fully wired. If control leaves the scope early (exception or early return) and the guard is still active, it restores the widget to full opacity (1.0) in the destructor. After the animation has been started and its own finished-handler will handle restoration, call release_() to disable the guard.

Note
The guard holds a QPointer to avoid dereferencing a deleted widget. If the widget is destroyed before scope exit, no action is taken.
Warning
Construct and use from the GUI thread only. The opacity change affects GUI state.

bool npi::StageWindow::OpacityGuard::active_ {true}

Whether the guard is active.

If true at destruction time and the widget pointer is valid, the guard will restore opacity to 1.0. Calling release_() sets this to false to make the destructor a no-op regarding opacity restoration.
