Detailed Description

template<typename T>
struct npi::RefPtrHost< T >::Obs

Inner observer type holding a type-erased pointer to the object. Used to broadcast the lifetime of the main object to RefPtr objects.

Note
  • Holds a std::shared_ptr<std::atomic<void*>> to ensure thread-safe access to a type-erased pointer.
  • Type-erased pointer is used instead of actual object pointer to allow for easy casting and polymorphic relations. To obtain the actual pointer, the RefPtr objects have to use static_cast<T*> which is done in RefPtr<T>::get_().

This struct object holds the type-erased pointer to the object. In its destructor the pointer is set to nullptr using std::memory_order_release to ensure thread-safe access. Since the object of this struct is stored as a std::shared_ptr<Obs>, once the last reference to std::shared_ptr<Obs> is destroyed, the destructor will be called and the object in the std::atomic<void*> will be replaced with nullptr. This allows the RefPtr objects to check if the object can still be dereferenced or not.
