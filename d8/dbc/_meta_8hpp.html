Detailed Description

Sole container file of npi::meta namespace.

This file contains the definition of the npi::meta namespace. This namespace holds constant organization and application metadata such as names, version, directory names, etc.

Note
  • This file is only supposed to have npi::meta namespace, do not add other namespaces, functions, classes, default values, etc. here.
  • Do not extend this namespace in other files. Meta.hpp is the only place where this namespace is defined.
