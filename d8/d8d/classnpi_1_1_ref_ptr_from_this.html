Detailed Description

template<typename T>
class npi::RefPtrFromThis< T >

Enables objects that inherits from this class to produce a RefPtr object referencing themselves if the object is wrapped in a RefPtrHost.

  • Behaves like a std::enable_shared_from_this but for RefPtrHost.
  • Stores a weak reference to the universal observer in the RefPtrHost.
  • Provides a way to obtain a RefPtr object to itself using the universal observer if the RefPtrHost object is still alive.
Template Parameters
TThe type of the object being wrapped in RefPtrHost and supporting enable-from-this semantics.
Note
This class only holds a weak reference to the universal observer in the RefPtrHost. This means that as long as any RefPtrHost object is alive, the RefPtrFromThis object will be able to create a RefPtr object to itself and that RefPtr object will be valid as long as the RefPtrHost object is alive.
Example of using LinkFromThis:
struct MyClass : RefPtrFromThis<MyClass> {
void hello() {}
};
auto host = RefPtrHost<MyClass>(new MyClass());
auto refPtr = host.universal_()->refPtrFromThis_();
if(refPtr) refPtr->hello();
