Detailed Description

Header-only utility for showing a top-level widget either instantly or with a source-origin animation.

This header defines npi::StageWindow, a helper class for top-level QWidget display:

  • quick_() positions and shows immediately.
  • animate_() creates a temporary frameless overlay window that grows/moves from a source rectangle (cursor or a "caller" widget) to the target widget's frame geometry, while the real widget fades from 0.0 to 1.0 opacity at the end.

Positioning guarantees:

  • The target widget is adjusted to remain inside the available geometry of the chosen screen.
  • Optional centering over a "center" widget.

Safety and lifetime notes:

  • animate_() does not capture this; it captures a QPointer<QWidget> to avoid use-after-free.
  • If the target widget is deleted during the animation, the temporary overlay is still cleaned up.
Warning
Call these APIs from the GUI thread. Showing/moving widgets from worker threads is undefined.
