Detailed Description

Namespace for directory paths used by the application.

This namespace provides functions to retrieve and manage directory paths. Functions in this namespace ensure that the directories exist, creating them if necessary.

◆ appProgramData()

const QString & npi::dirs::appProgramData ( )
nodiscardnoexcept

Absolute path to C:\ProgramData\[orgFolder]\[appFolder].

This function does not create C:\ProgramData itself; if it is missing. It only creates the [orgFolder] and/or [appFolder] subfolders if not already present.

Returns
Absolute path on success; empty QString on failure.
Warning
Caller MUST check for empty QString, if true, treat it as error.
Note
  • Depends on orgProgramData(). If that fails, this also fails.
  • On non-Windows builds this returns empty QString.
