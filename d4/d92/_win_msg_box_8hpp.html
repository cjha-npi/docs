Detailed Description

Windows message box helpers using the Unicode API.

This header provides thin, dependency-light helpers around the Win32 MessageBoxW function. They can be called safely even if no Qt GUI infrastructure (QApplication) has been created yet, making them suitable for early-startup, late-shutdown, and fatal-error reporting paths.

Design goals:

  • Always use the Unicode (wide) Windows API to avoid code page issues.
  • Keep behavior predictable: choose an appropriate modality when the caller does not specify one, and foreground the dialog when no owner is available to reduce Z-order surprises.
  • Overload to accept optional QWidget owner, converting to HWND.
  • Avoid throwing exceptions; all functions are noexcept.

Notes:

  • NOMINMAX is defined to prevent macro collisions with std::min/stdmax.
  • WIN32_LEAN_AND_MEAN is defined to reduce compile-time overhead.
