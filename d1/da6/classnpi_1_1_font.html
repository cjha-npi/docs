Detailed Description

Thread-safe font utility with style, sizing, and caching.

The Font class exposes:

  • Global setters for the base application font and the base fixed-width font.
  • A Style struct to describe per-request font attributes.
  • Absolute and relative point-size inputs with effective-size calculation.
  • Cached Groups of (QFont, QFontMetrics, QFontMetricsF) for fast reuse.
  • Static mutexes to protect shared state and caches.

Typical use:

// Initialize at app startup (GUI thread):
npi::Font::SetAppFont_(qApp->font());
// On demand, configure style and size:
npi::Font::Style st;
st.bold_ = true;
npi::Font f{st};
f.setRelativeSize_(+1);
// Fetch QFont or metrics:
QFont font = f.appFont_();
QFontMetrics fm = f.appFontMetrics_();

Get or create the cached Group for the application font family.

Retrieves or creates a cached Group for the application font. The effective font size is computed based on either the absolute size (if > 0) or the current application font size plus the relative size adjustment. A hash key is computed from the style and size, and the cache is checked for an existing entry. If not found, a new QFont is configured, stored in a Group, and inserted into the cache.

Parameters
styleStyle flags and capitalization.
absSizeAbsolute point size (> 0 overrides).
relSizeRelative point size delta.
Returns
Reference to cached Group (valid for the lifetime of the cache).
Note
Uses AppFont_ as the base family.

inlinestaticprivate

Compute a hash key from style and size.

Combines boolean flags, capitalization, and size into a single 32-bit integer. Bit layout before qHash:

  • bits 0..4 : bold, italic, underline, overline, strikeout
  • bits 5..12 : capitalization (8 bits)
  • bits 13..28: point size (16 bits, truncated, assumes size is within [0, 65535])
Parameters
styleStyle flags and capitalization.
sizePoint size used for the key (truncated to 16 bits).
seedOptional hash seed.
Returns
size_t hash suitable as a QHash key.
