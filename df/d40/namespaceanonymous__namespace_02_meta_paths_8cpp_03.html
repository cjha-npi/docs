Detailed Description

Anonymous namespace for internal usage.

This namespace contains helper functions used for meta path resolution and creation. These functions are used internally within this translation unit by

  • npi::app::orgProgramDataDir()
  • npi::app::orgPublicDocumentsDir()
  • npi::app::appProgramDataDir()
  • npi::app::appPublicDocumentsDir()
    Note
    This namespace is anonymous to limit the scope of the function it contains to this translation unit only. Since these functions terminate the process on failure, they should not be called from outside this translation unit.

Resolve base path using Known Folder ID or environment variable fallback.

This functions attempts to resolve the base path for the given folder ID by first querying the Windows API and falling back to an environment variable if necessary.

Parameters
folderIdThe Known Folder ID to query via SHGetKnownFolderPath.
envVarThe environment variable to check if the API call fails.
envSuffixThe suffix to append to the environment variable path.
pathNameForErrorsThe name of the path for error messages.
Warning
On unrecoverable failure (cannot locate or validate the directory), a native dialog is shown and the process is terminated (no return).
