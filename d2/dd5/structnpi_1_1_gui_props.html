Detailed Description

Scope-only container for GUI property keys.

GuiProps provides a closed set of GUI-related dynamic property names for use with QObject. The type cannot be constructed, copied, moved, or destroyed. Only the static members are intended to be used. All keys are ASCII-only and stable by contract so they can be relied upon by styles, delegates, and other UI infrastructure without string duplication or translation.

Invariants:

  • No instances of GuiProps exist at runtime.
  • All keys are null-terminated and suitable for QObject::setProperty().
  • Keys are intended to be read-only and never modified.

Extension policy:

  • Add new keys as additional static members in this type to keep a single source of truth and predictable auto-complete. If third-party modules must contribute keys without editing this file, prefer a namespace container.
