Detailed Description

Root namespace for the application.

Contains all public APIs and common types. It includes:

  • npi::app module-agnostic identifiers, compile-time defaults, and lightweight utilities.
  • Feature-specific child namespaces (e.g., npi::log), each owning its data, types, and helpers.
  • Core application components and classes.

Use npi::app for cross-cutting code; place domain-specific code in a dedicated child namespace.
