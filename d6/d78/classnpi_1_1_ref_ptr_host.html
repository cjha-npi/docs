Detailed Description

template<typename T>
class npi::RefPtrHost< T >

A thread-safe, owning host that manages an object of type T and notifies RefPtr objects upon destruction.

  • Behaves like a std::shared_ptr i.e. owns the object T and destroys it when the last reference is released.
  • Provides direct pointer-like access to the object.
  • Supports polymorphic conversion and comparison.
  • Maintains two RefPtr channels: Universal and Local.
  • RefPtr object obtained through Universal channel is valid while the object is alive.
  • RefPtr object obtained through Local channel is valid only while the last local copy of RefPtrHost is alive.
Template Parameters
TThe type of the object being hosted.
Example of creating and using links:
RefPtrHost<MyClass> host(new MyClass());
auto refPtr = host.universal_();
if (refPtr) refPtr->doSomething_();

Note
Private function for internal use only.
  1. If identical, return.
  2. If other.get_() non-null:
