Detailed Description

Namespace for metadata.

This namespace contains constant organization and application metadata such as names, version, directory names, etc.

Note
  • This namespace is only supposed to have constant metadata values, do not add functions, classes, default values, etc. here.
  • Do not extend this namespace in other files. Meta.hpp is the only place where this namespace is defined.
  • Keep values ASCII/safe and compile time constant (constexpr) where possible.
See also
  • npi::defs for shipped default values.
  • npi::path for file/folder paths.
  • npi::util for utility functions.
