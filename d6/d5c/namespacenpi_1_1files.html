Detailed Description

Namespace for file paths used by the application.

This namespace provides functions to retrieve file paths used by the application. Functions in this namespace return file paths without creating the files.

const QString & npi::files::settings ( )
nodiscardnoexcept

Absolute path of application's main settings.ini at C:\ProgramData\[orgFolder]\[appFolder]\.

This function does not create C:\ProgramData itself; if it is missing. It only creates the [orgFolder] and/or [appFolder] subfolders if not already present.

Returns
Absolute path on success; empty QString on failure.
Warning
Caller MUST check for empty QString, if true, treat it as error.
Note
  • This function does not create the settings.ini file itself.
  • Depends on appProgramData(). If that fails, this also fails.
  • On non-Windows builds this returns empty QString.
