1/**
2* \file Props.hpp
3* \brief Dynamic property keys used with QObject and QWidget for styling and behavior.
4*
5* \details
6* These string constants are the names of dynamic properties that can be attached to
7* any QObject (including QWidget and subclasses) using QObject::setProperty() and
8* read using QObject::property(). They are primarily consumed by a QProxyStyle
9* implementation and related helpers to adjust rendering, hit-testing, and layout
10* without subclassing each control.
11*
12* All properties are optional. When a property is missing, the default Qt behavior
13* should apply. Values are stored as QVariant; use the value types indicated in each
14* entry below.
15*
16* Typical usage:
17* auto* btn = new QPushButton("Apply", parent);
18* btn->setProperty(npi::props::PushButtonRightIcon, true);
19* btn->setProperty(npi::props::IconName, "apply");
20*
21* See also: QObject::setProperty, QObject::property, QVariant, QStyle, QProxyStyle
22*/
