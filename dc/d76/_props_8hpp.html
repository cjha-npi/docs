Detailed Description

Dynamic property keys used with QObject and QWidget for styling and behavior.

These string constants are the names of dynamic properties that can be attached to any QObject (including QWidget and subclasses) using QObject::setProperty() and read using QObject::property(). They are primarily consumed by a QProxyStyle implementation and related helpers to adjust rendering, hit-testing, and layout without subclassing each control.

All properties are optional. When a property is missing, the default Qt behavior should apply. Values are stored as QVariant; use the value types indicated in each entry below.

Typical usage: auto* btn = new QPushButton("Apply", parent); btn->setProperty(npi::props::PushButtonRightIcon, true); btn->setProperty(npi::props::IconName, "apply");

See also: QObject::setProperty, QObject::property, QVariant, QStyle, QProxyStyle
