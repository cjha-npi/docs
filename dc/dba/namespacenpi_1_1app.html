Detailed Description

Application metadata and constants.

This namespace contains compile-time constants and identifiers related to the application, such as organization name, application name, version, and directory names.

These constants are defined as QLatin1StringView for efficiency and ease of use in various parts of the application, including settings management, logging, and user interface elements.
