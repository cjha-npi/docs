Detailed Description

Namespace for metadata and defaults.

This namespace is for these two types of data:

  • Application metadata (e.g. app name, version, organization name, etc.)
  • Default values (e.g. default file names, default settings, etc.)
Note
  • This namespace is only supposed to have constant variables, do not add functions or classes here.
  • Do not extend this namespace in other files. AppData.hpp is the only place where this namespace is defined.
See also
npi::util for utility functions.
