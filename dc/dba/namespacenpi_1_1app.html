Detailed Description

Application metadata and constants.

This namespace contains compile-time constants and identifiers related to the application, such as organization name, application name, version, and directory names.

These constants are defined as QLatin1StringView for efficiency and ease of use in various parts of the application, including settings management, logging, and user interface elements.

const QString & npi::app::appProgramDataDir ( )
nodiscardnoexcept

Absolute path to C:\ProgramData\[orgFolder]\[appFolder].

This function does not create C:\ProgramData itself; if it is missing. It only creates the [orgFolder] and/or [appFolder] subfolders if not already present.

Returns
Absolute path on success; empty QString on failure.
Warning
Caller MUST check for empty QString, if true, treat it as error.
Note
  • Depends on orgProgramData(). If that fails, this also fails.
  • On non-Windows builds this returns empty QString.
