1/**
2 * \file GuiProps.hpp
3 * \brief Static container of GUI-related QObject dynamic property keys.
4 * \details
5 * This header defines a non-instantiable utility type that exposes a fixed set
6 * of null-terminated C-string property names. These names are intended to be
7 * used with QObject::setProperty() and QObject::property() across the GUI
8 * layer. Each key is a stable ASCII identifier, designed for use by QProxyStyle
9 * customizations and other rendering or behavior hooks.
10 *
11 * Design notes:
12 * - Keys are provided as static constexpr character arrays. This gives exact
13 * storage with static lifetime and no heap allocation. The array decays to
14 * const char* when passed to Qt APIs.
15 * - The type is sealed and cannot be instantiated. It is a named scope only.
16 * - Keys follow your static-member naming rule: capitalized and ending with "_".
17 *
18 * Usage example:
19 * \code
20 * // Set and read a dynamic property using the stable key
21 * obj->setProperty(npi::GuiProps::IconName_, "settings");
22 * const QVariant v = obj->property(npi::GuiProps::TransparentForMouseWheel_);
23 * \endcode
24 */

31 /**
32 * \brief Scope-only container for GUI property keys.
33 * \details
34 * GuiProps provides a closed set of GUI-related dynamic property names for use
35 * with QObject. The type cannot be constructed, copied, moved, or destroyed.
36 * Only the static members are intended to be used. All keys are ASCII-only and
37 * stable by contract so they can be relied upon by styles, delegates, and other
38 * UI infrastructure without string duplication or translation.
39 *
40 * Invariants:
41 * - No instances of GuiProps exist at runtime.
42 * - All keys are null-terminated and suitable for QObject::setProperty().
43 * - Keys are intended to be read-only and never modified.
44 *
45 * Extension policy:
46 * - Add new keys as additional static members in this type to keep a single
47 * source of truth and predictable auto-complete. If third-party modules must
48 * contribute keys without editing this file, prefer a namespace container.
49 */
